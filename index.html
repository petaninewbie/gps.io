Monitor GPS Real-Time

Aplikasi web ini dapat diakses di Android dan iPhone.

Monitoring Non-aktif

Pengaturan Pembaruan (GPS Options)

Pengambilan data GPS diatur oleh opsi **High Accuracy** dan pembaruan paksa **Maximum Age: 1 detik**.

Timeout Aktif: 20 detik (Waktu maksimum untuk mendapatkan sinyal).

Tindakan Diperlukan: Berikan Izin Lokasi!

Kesalahan "Akses lokasi ditolak" berarti peramban Anda memblokir sensor GPS. Anda perlu **memberikan izin lokasi** di pengaturan peramban (cari ikon gembok di bilah alamat, atau cek notifikasi perizinan). Setelah diizinkan, klik tombol **Aktifkan Monitoring** lagi.
